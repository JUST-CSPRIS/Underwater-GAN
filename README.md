提出一种用于水下图像增强的退化感知深度展开网络。受深度展开方法启发，提出用于图像增强的展开增强模块，将图像映射到特征空间实现迭代增强，并通过可学习的注意力机制参数化步长与动量项，优化迭代的收敛速度。在每个展开增强模块中构造退化感知正则模块，将物理退化特性和散射噪声抑制显式嵌入深度特征空间。并结合下采样特征融合策略，实现高分辨率纹理细节与低分辨率语义上下文的联合学习，减轻了水下退化环境的负面影响。训练过程采用多损失函数联合优化策略，满足不同阶段网络的需求。实验结果标明，在UFO-120数据集中，所提网络的PSNR指标达到26.51dB，SSIM指标达到0.7885。在其他公开数据集上的实验结果也验证了该网络在水下图像增强任务中的优越性。
